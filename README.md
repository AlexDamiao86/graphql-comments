# RocketComments

RocketComments offers an easy way to post and check comments using a GraphQL API. 
Application built during a Rocketseat's live session. 

## Built With

* [GraphQL]
(https://graphql.org) 

* [Apollo-Server] 
* [Appollo-Client] 
(https://www.apollographql.com/docs/)
